React-powered addons for Odoo

What it does

SC React Theme maps shadcn/zinc design tokens from the Mantine theme system onto Odoo's SCSS variables — so React-powered widgets and standard Odoo forms, lists, and buttons share one cohesive visual language.

SC React Theme Features

A clean zinc palette, applied once at the variable layer — no component overrides, no upgrade headaches.

Zinc Design Tokens

Applies the shadcn/zinc design tokens — a clean, modern neutral palette with carefully balanced grays for backgrounds, borders, and text. 48 SCSS variables mapped from the shared shadcn-theme.ts SSOT.

Unified React + Native Styling

React components rendered by odoo-react and standard Odoo views share one design language — no visual seam between framework widgets and native screens.

Zero Component Overrides

Only global SCSS variables are overridden — no component-level CSS. Future Odoo upgrades are unlikely to conflict, and custom modules keep working unchanged.

Built on

Powered by the odoo-react bridge and the shadcn/zinc design system — running natively on Odoo.
